## Unmanaged Files

- `~/.ssh`: contains ssh public and private keys
- `~/.gnupg`: contains gpg public and secret keys
- `~/.abook/addressbook`: contains contact data
- `./newsboat/urls`: contains rss feeds
- `~/smb-secrets`: contains samba login details
